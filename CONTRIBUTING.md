# Contributing to MDTI

感谢你对 MDTI 的兴趣！欢迎提 Issue 和 PR。

## 贡献方向

- 🐛 **Bug 修复**：发现页面异常、计算错误、显示问题
- ✍️ **题目/文案优化**：让题目更贴近真实职场，人格描述更有共鸣
- 🎨 **UI 改进**：移动端体验、动画、无障碍访问
- 🌐 **国际化**：英文版、繁中版

## 提 Issue

- Bug：请附上浏览器/系统版本，截图或复现步骤
- 内容建议：直接描述你觉得哪里不够准确/有趣

## 提 PR

1. Fork 本仓库
2. 在 `index.html` 中修改（所有代码都在这一个文件里）
3. 本地用 `npx serve .` 或 `python3 -m http.server` 验证
4. 提 PR，描述改了什么、为什么

## 内容原则

- 幽默但不冒犯特定群体
- 自嘲式，不是攻击式
- 有共鸣，打工人看了会说"就是我"
- 不涉及心理学专业术语，避免误导
